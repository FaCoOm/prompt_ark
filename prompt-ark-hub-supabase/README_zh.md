## 😫 问题

你在 Reddit 上找到了一个超棒的提示词扔进笔记...然后再也没用过。团队同事天天问你要同样的提示词。最好的提示词散落在各地。

**提示词方舟社区解决这些问题。**

## ✨ 功能一览

| 你想要... | 提示词方舟社区提供... |
|---|---|
| **寻找提示词** | 🔍 关键词搜索，按分类筛选 |
| **发现热门** | 🔥 查看社区最爱的内容 |
| **即时预览** | 📖 精美的 Markdown 渲染 |
| **在聊天中使用** | ⚡ 一键安装到提示词方舟扩展 |
| **收藏备用** | 🍴 复制任意提示词到个人收藏 |
| **贡献社区** | 🌟 与社区分享你的最佳提示词 |

## 🔌 与提示词方舟扩展配合使用

社区设计为与你的提示词方舟扩展完美配合：

1. **浏览** → 在社区找到一个很棒的提示词
2. **安装** → 点击**添加到提示词方舟** → 直接发送到你的扩展
3. **使用** → 打开 ChatGPT/Claude/Gemini → 选择提示词 → 自动注入聊天框

你的扩展和社区是互联互通的。你保存的提示词也可以分享回社区。

## 🛠 面向开发者

- **前端**: React 19 + TypeScript + Vite
- **后端**: Supabase (PostgreSQL + 认证)
- **样式**: CSS 变量 + 自定义组件
- **Markdown**: react-markdown + remark-gfm
