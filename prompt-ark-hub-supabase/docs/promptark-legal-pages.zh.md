# /privacy

**PromptArk 隐私政策**

**生效日期：** 2026 年 3 月 25 日  
**最后更新：** 2026 年 3 月 27 日

PromptArk 是由 **OOMeta** 运营的产品。其运营主体为 **深圳无限元智能网络有限公司**（以下简称“OOMeta”“我们”或“本公司”）。本隐私政策说明，当用户访问或使用 PromptArk 网站、Hub、浏览器扩展及相关服务（以下统称“服务”）时，我们如何收集、使用、存储、披露和保护信息。

## 1. 适用范围

本隐私政策适用于：
- PromptArk 网站和 Hub；
- PromptArk 浏览器扩展；
- Google OAuth 身份验证和账户登录；
- 与用户账户关联的同步和存储功能；以及
- 相关的支持与账户管理活动。

## 2. 我们收集的信息

我们仅收集在合理范围内为运营和提供服务所必需的信息。

### 2.1 Google OAuth 账户信息
当用户通过 Google OAuth 登录时，我们可能收集：
- 电子邮箱地址；
- 显示名称；以及
- 个人资料图片 / 头像。

我们使用这些信息对用户进行身份验证、创建或维护账户，并提供与账户关联的功能。

### 2.2 用户内容
我们会收集并存储用户自愿通过 PromptArk 创建、保存、整理、同步或管理的内容，包括：
- prompts；
- prompt 标题、标签、分类、元数据和设置；
- 用户创建的 prompt 模板及相关配置数据；以及
- 扩展为用户生成并保存在本地或同步到 Hub 的内容。

### 2.3 浏览器扩展与页面交互数据
根据用户所调用的功能，PromptArk 浏览器扩展可能访问：
- 当前活动标签页；
- 当前页面 URL；
- 页面标题；
- 选中文本；
- 页面文本或页面上下文；
- 当用户使用相关功能时基于剪贴板的上下文；以及
- 用于 prompt 注入或用户请求的 prompt 操作的输入框或 AI 平台编辑器。

此类访问仅限于启用扩展已披露、面向用户且由用户主动触发的功能，例如 prompt 插入、上下文捕获、智能转换、文章分享以及其他由用户发起的 prompt 相关工作流。

### 2.4 本地存储与同步数据
PromptArk 可能将数据存储于：
- 浏览器本地；
- 受支持的浏览器存储或同步功能中；以及
- 当用户登录并使用同步功能时，我们的后端系统中，包括 Supabase。

### 2.5 技术与运营数据
我们可能处理为维护服务所必需的有限技术数据，例如：
- 身份验证事件；
- 同步状态；
- 调试所需的错误信息；
- 基本请求元数据；以及
- 服务配置信息。

目前，我们不会通过第三方分析或崩溃报告平台有意收集分析数据或崩溃报告数据。

## 3. 我们如何使用信息

我们仅将信息用于合法的产品和运营目的，包括：
- 对用户进行身份验证并维护账户；
- 提供 Hub 和浏览器扩展功能；
- 保存、整理、同步和展示用户的 prompts；
- 按用户请求将 prompts 插入受支持的 AI 平台；
- 为用户请求的 prompt 功能捕获页面上下文、选中文本或相关内容；
- 为用户生成、优化、翻译或转换 prompt 内容；
- 维护、保护、调试并改进服务；
- 响应支持请求；
- 执行我们的服务条款；以及
- 遵守适用法律。

我们**不会**向第三方出售个人信息或用户内容。

我们**不会**将个人信息、浏览活动或用户内容用于定向广告。

## 4. 浏览器扩展数据实践

PromptArk 浏览器扩展旨在帮助用户在受支持的网站和 AI 平台中管理和使用 prompts。

扩展仅在为提供面向用户、可见且由用户主动请求的功能所必需的范围内与网页内容交互。这可能包括：
- 检测受支持的 AI 平台并定位输入区域；
- 将 prompts 注入编辑器或文本字段；
- 捕获选中文本或页面上下文，用于 prompt 生成或 prompt 变量填充；
- 启用文章分享或上下文感知的 prompt 工作流；以及
- 将生成的 prompt 内容存储在本地或同步到用户的 PromptArk 账户。

PromptArk **不会**出售浏览活动数据。

PromptArk **不会**将浏览活动用于广告、画像分析或转售。

除为提供产品界面及适用商店列表中所描述的面向用户功能所严格必需的范围外，PromptArk **不会**收集或保留用户的网页浏览历史。

## 5. Cookies 与会话处理

PromptArk 本身不会利用浏览器扩展的权限通过 cookies 出售、跟踪或画像分析用户。

在技术上确有必要时，与 cookies 或会话处理相关的扩展权限仅可用于支持已认证的产品功能或兼容集成，例如依赖现有浏览器会话的用户请求 AI 服务交互。

## 6. 信息共享

我们不会出售或出租个人信息。

我们仅可能在以下有限情形下共享信息：
- 与 **Google** 共享，以处理 Google OAuth 身份验证和登录流程；
- 与 **Supabase** 共享，作为我们的后端服务提供商，用于身份验证、数据库、存储和同步功能；
- 与**托管和基础设施提供商**共享，包括我们用于运行 Hub 及相关服务的云托管和服务器运营商；
- 与**电子邮件和通信服务提供商**共享，以在需要时响应支持或运营消息；
- 与**法律或监管机构**共享，在适用法律或有效法律程序要求下；以及
- 与**继承方或关联方**共享，在合并、收购、重组、融资或资产出售过程中，并受适当法律保护约束。

我们不会为第三方自身的广告或独立营销目的共享用户数据。

## 7. 数据保留

我们仅在为以下目的而合理必要的期间内保留个人信息和用户内容：
- 提供服务；
- 维护用户账户和同步功能；
- 解决争议；
- 执行我们的协议；以及
- 履行法律义务。

如果用户请求删除账户及相关数据，我们将在商业上合理的期限内删除或匿名化相关信息，除非法律、安保需求、欺诈防范或正当争议解决目的要求保留。

## 8. 用户权利与选择

根据适用法律，用户可能享有以下权利：
- 访问其个人信息；
- 请求更正不准确的信息；
- 请求删除其账户及相关数据；
- 反对或限制某些处理；以及
- 在同意是处理法律依据时撤回同意。

用户可通过联系 **privacy@oometa.ai** 请求删除其账户及相关数据。

## 9. 国际用户

PromptArk 面向全球用户提供服务。信息可能在用户所在国家或地区之外的司法辖区进行处理。在适用法律要求的情况下，我们将采取合理措施，为跨境数据处理提供适当保障。

## 10. 安全

我们采取合理的管理、技术和组织措施，以保护个人信息和用户内容免遭未经授权的访问、丢失、篡改、披露或滥用。

但是，没有任何传输或存储方式是绝对安全的，我们无法保证绝对安全。

## 11. 儿童隐私

本服务并非面向 13 岁以下儿童，或适用法律要求的更高最低年龄以下的未成年人。我们不会在违反适用法律的情况下故意收集儿童的个人信息。

## 12. 第三方服务

本服务可能与第三方平台和服务集成或依赖其提供支持，包括 Google OAuth、受支持的 AI 平台、云托管服务提供商、域名和电子邮件服务提供商以及 Supabase。该等第三方可能依据其自身条款和隐私政策运营。

## 13. 有限使用披露

PromptArk 对从 Google APIs 接收的信息的使用将遵守 **Google API Services User Data Policy**，包括 **Limited Use** 要求。

对于 PromptArk 浏览器扩展，我们对个人或敏感用户数据的使用仅限于提供或改进扩展的单一目的及其面向用户的功能，相关内容以产品界面和适用商店列表中的说明为准。我们不会出售此类数据，不会将其用于广告，也不会允许人工读取此类数据，除非出于安全、法律合规需要，或获得用户明确同意。
