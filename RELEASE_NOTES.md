# Prompt Ark V2 Update Notes

## Chrome Web Store short summary

Prompt Ark V2 upgrades prompt organization and model support: a new prompt taxonomy with system/custom categories, smarter AI classification, and additional model providers including DeepSeek Web, Kimi, Qwen, Grok, GLM, Gemini API, and OpenAI-compatible APIs.

## 中文上架更新摘要

Prompt Ark V2 重点升级了提示词工程管理体验：

- 重构提示词分类体系，支持系统分类、自定义分类、AI 推荐分类和待确认状态。
- 新增/增强多模型接入，除默认 Gemini Web 外，可接入 DeepSeek Web、Kimi、Qwen、Grok、GLM、豆包、ChatGPT Web，以及 Gemini API 和 OpenAI 兼容接口。
- 智能转换、自动标题/标签、导入和分享流程现在会携带更完整的分类与输出类型信息。
- 优化提示词库浏览、筛选、分类确认和同步体验，让大量提示词更容易长期维护。

## English release summary

Prompt Ark V2 focuses on a cleaner prompt-engineering workflow:

- Rebuilt prompt taxonomy with system categories, custom categories, AI recommendations, and category review state.
- Expanded model support beyond the default Gemini Web provider, including DeepSeek Web, Kimi, Qwen, Grok, GLM, Doubao, ChatGPT Web, Gemini API, and OpenAI-compatible APIs.
- Smart Convert, auto metadata extraction, import, and sharing now preserve richer category and output-modality metadata.
- Improved browsing, filtering, review, and sync behavior for larger prompt libraries.
