# 全部功能一览

## 🎯 核心三板斧（从这里开始）

### ✨ 提示词选择器 — 你的提示词库触手可及
在任何支持的 AI 平台，点击聊天框旁的 **✨** 按钮：
- 搜索、按分类浏览、Markdown 渲染预览
- 开箱即带 100 条内置提示词
- 含 `{{变量}}` 的提示词会弹出填写表单

### ⚡ 快捷指令 — 一键文字加工
点击聊天框旁的 **⚡** 按钮：

| 操作 | 效果 |
|---|---|
| ✏️ 润色 | 优化表达，提升专业感 |
| 📋 总结 | 提炼核心要点 |
| ➕ 扩写 | 增加深度和细节 |
| 🌐 翻译 | 翻译为英文 |
| 💡 解释 | 用大白话讲清概念 |

> 这些功能 **不需要 API Key** — 它们把指令注入聊天框，让平台自己的 AI 处理。

### `/斜杠` 命令 — 更快
在任何聊天框输入 `/` + 关键词，下拉匹配列表出现，选中即展开。不需要打开弹窗。

## 🔧 进阶功能

### 动态变量 (`{{var}}`)
在提示词正文中写 `{{主题}}` 或 `{{语言}}`，使用时自动弹出表单逐个填写。适合做可复用模板。

### AI 提示词优化器
编辑提示词 → 点 **✨ 优化** → 生成 3 个版本：
1. **精简版** — 只留精华
2. **增强版** — 补充约束和格式
3. **专业版** — 完整角色设定 + 思维链

逐行 Diff 对比视图。一键采纳或放弃。

### 🌐 AI 提示词翻译
一键将任意提示词翻译成 7 种语言：
1. 点击提示词卡片上的 **🌐 地球仪** 图标按钮
2. 或在编辑弹窗中使用 **翻译** 功能
3. 选择目标语言：English · 中文 · 日本語 · Español · Français · Deutsch · 한국어
4. 标题、分类、标签和正文内容全部翻译，**自动保存**

> 自动保留 `{{变量}}`、Markdown 格式和技术术语。默认使用 Gemini Web（无需 API Key）。如果 Gemini 会话过期，会自动打开登录页面。

### 右键即存
网页上选中文字 → 右键 → **"添加到 Prompt Ark"**。AI 自动生成标题、分类和标签。

### 键盘快捷键
`Ctrl+Shift+P`（Mac 为 `Cmd+Shift+P`）在任何页面唤出选择器。

### 版本历史与回滚
每次编辑自动存档旧版本。点 **历史** 浏览、Diff 对比、一键恢复。

### 收藏与智能筛选
- ❤️ 点心形图标收藏
- 侧栏过滤：**全部** · **收藏** · **最近** · **最常用**

## 📦 分享与社区

### 分享提示词
点分享图标 → 创建 GitHub Gist → 生成 **Prompt Hub** 展示页链接。可发 Twitter、Reddit 或复制 URL。

### 提示词包 (Prompt Pack)
多选提示词 → Pack 模式 → 打包发布为合集。

> 分享功能需要在 设置 → 同步 中配置 GitHub Token。

---

## 📥 导入提示词

### 粘贴导入
直接粘贴原始文本或 JSON。

### URL 导入
输入网址（如 GitHub awesome-prompts 仓库），插件会：
1. 爬取页面解析提示词
2. AI 质量评分打分
3. 按最低分数线筛选

> 💡 配置 GitHub Token 可避免速率限制（5000 次/小时 vs 60 次/小时）。

## 🧩 更多好玩的

### 页面上下文魔法变量
| 变量 | 自动填充为 |
|---|---|
| `{{page_title}}` | 当前标签页标题 |
| `{{page_url}}` | 当前标签页 URL |
| `{{selected_text}}` | 页面上选中的文字 |
| `{{page_text}}` | 整页文字内容 |

### 侧边栏模式
Prompt Ark 可作为 **侧边栏** 常驻浏览器右侧，点击工具栏图标切换。
