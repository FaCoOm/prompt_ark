# 全部功能一览

## 🎯 核心三板斧（从这里开始）

### ✨ 提示词选择器 — 你的提示词库触手可及
在任何支持的 AI 平台，点击聊天框旁的 **✨** 按钮：
- 搜索、按分类浏览、Markdown 渲染预览
- 开箱即带 100 条内置提示词
- 含 `{{变量}}` 的提示词会弹出填写表单

### ⚡ 快捷指令 — 一键文字加工
点击聊天框旁的 **⚡** 按钮：

| 操作 | 效果 |
|---|---|
| ✏️ 润色 | 优化表达，提升专业感 |
| 📋 总结 | 提炼核心要点 |
| ➕ 扩写 | 增加深度和细节 |
| 🌐 翻译 | 翻译为英文 |
| 💡 解释 | 用大白话讲清概念 |

> 这些功能 **不需要 API Key** — 它们把指令注入聊天框，让平台自己的 AI 处理。

### `/斜杠` 命令 — 更快
在任何聊天框输入 `/` + 关键词，下拉匹配列表出现，选中即展开。不需要打开弹窗。

## 🔧 进阶功能

### 动态变量 — 三种形式
Prompt Ark 支持三种变量写法，使用时对应不同的交互控件：

| 写法 | 控件类型 | 适用场景 |
|---|---|---|
| `{{主题}}` | 自由输入 | 任意开放内容 |
| `{{语言:中文|英文|日文}}` | 下拉枚举 | 固定选项集合 |
| `{{风格:正式}}` | 预填文本 | 有合理默认、用户可改 |

### 提示词分类体系
Prompt Ark 使用结构化元数据管理提示词。现在每条提示词可以同时拥有：

| 元数据 | 作用 |
|---|---|
| **输出类型** | 标记为文本、图像或视频提示词 |
| **系统分类** | 10 个稳定领域：通用与效率、写作与编辑、营销与品牌、销售与客服、商业与运营、研究与学习、编程与开发、数据与分析、设计与视觉、创意与媒体 |
| **自定义分类** | 你自己的分类标签，可在新建/编辑时复用 |
| **AI 推荐分类** | 保存、导入、智能转换时由当前模型自动推荐 |
| **待确认状态** | 当 AI 置信度不足时，卡片会提示先确认分类 |

编辑提示词时可以在 **系统分类 / 自定义分类 / 新建** 之间切换。待确认的提示词会显示“分类待确认”，点进去后可以使用当前分类、采纳 AI 推荐，或改成你自己的分类。

### 上下文变量（`{{@...}}`）
自动解析当前标签页内容，无需手动复制粘贴：

| 变量 | 自动填充为 |
|---|---|
| `{{@page_title}}` | 当前标签页标题 |
| `{{@page_url}}` | 当前标签页 URL |
| `{{@selection}}` | 页面上选中的文字 |
| `{{@page_text}}` | 页面全文内容 |
| `{{@date}}` | 当天日期（YYYY-MM-DD） |
| `{{@lang}}` | 当前界面语言设置 |

> `{{@...}}` 变量在执行时始终读取当前标签页。

### AI 提示词优化器
编辑提示词 → 点 **✨ 优化** → 生成 3 个版本：
1. **精简版** — 只留精华
2. **增强版** — 补充约束和格式
3. **专业版** — 完整角色设定 + 思维链

逐行 Diff 对比视图。一键采纳或放弃。

### 🌐 AI 提示词翻译
一键将任意提示词翻译成 7 种语言：
1. 点击提示词卡片上的 **🌐 地球仪** 图标按钮
2. 或在编辑弹窗中使用 **翻译** 功能
3. 选择目标语言：English · 中文 · 日本語 · Español · Français · Deutsch · 한국어
4. 标题、分类、标签和正文内容全部翻译，**自动保存**

> 自动保留 `{{变量}}`、Markdown 格式和技术术语。默认使用 Gemini Web（无需 API Key）。如果 Gemini 会话过期，会自动打开登录页面。

### 🪄 智能转换
网页上选中任意文字 → 右键 → **"智能转换为提示词"**。

AI 分析选中内容、推断意图，生成完整的角色设定 + 任务描述，将下属实体提取为 `{{变量}}`，自动带标题、分类、标签入库。

> 默认使用 Gemini Web（无需 API Key）。适合把文章、报告、案例转成可复用模板。生成结果会同时包含标题、输出类型、系统/自定义分类建议和标签。

### 右键即存
网页上选中文字 → 右键 → **"添加到 Prompt Ark"**。AI 自动生成标题、输出类型、分类和标签。

### 键盘快捷键
`Ctrl+Shift+P`（Mac 为 `Cmd+Shift+P`）在任何页面唤出选择器。

### 抓取上下文（`Ctrl+Shift+G`）
将当前页面（标题、URL、选中文字、全文）转成一张智能 Prompt 卡片，并把该页面信息写入卡片的 `sourceContext` 供后续溯源查看。

> Mac 上为 `Cmd+Shift+G`

### 版本历史与回滚
每次编辑自动存档旧版本。点 **历史** 浏览、Diff 对比、一键恢复。

### 收藏与智能筛选
- ❤️ 点心形图标收藏
- 侧栏过滤：**全部** · **收藏** · **最近** · **最常用**

## 📦 分享与社区

### 分享提示词
点分享图标 → 发布到 **Prompt Hub** → 生成展示页链接。再配合 AI 生成各平台定制内容：

| 平台 | 内容风格 |
|---|---|
| **Twitter** | 简洁推文 + hashtag |
| **Reddit** | Markdown 标题 + 正文 |
| **知乎** | 中文长文专栏文章 |
| **微信** | 移动端友好的公众号文章 |
| **小红书** | 短图文 + Emoji + 标签 |

> 发布到 Hub 需要登录 Prompt Ark 账号。

### 提示词包 (Prompt Pack)
多选提示词 → Pack 模式 → 打包发布为合集。

> Prompt Pack 同样通过 Prompt Hub 发布，也需要登录账号。

## 📥 导入提示词

### 粘贴导入
直接粘贴原始文本或 JSON。

### URL 导入
输入文件链接，或 GitHub 仓库/目录链接，插件会：
1. 扫描可导入内容并解析提示词
2. AI 质量评分打分
3. 按最低分数线筛选

> 💡 导入 GitHub 链接时，可在 URL 导入页填写 GitHub Token，以减少限流（5000 次/小时 vs 60 次/小时）。
