# 30 秒极速上手

## 安装与固定（10 秒）
1. 下载或克隆仓库 → 打开 `chrome://extensions/`（或 `edge://extensions/`）→ 开启**开发者模式** → 点击**加载已解压的扩展程序** → 选择项目文件夹
2. 点击 🧩 拼图图标 → 📌 固定 Prompt Ark 到工具栏

> 💡 也可以当 **侧边栏** 用——点击图标直接停靠到浏览器侧边。

## 马上试试（20 秒）

装完 **立刻能用**，不需要 API Key，不需要任何配置。

1. 打开 [ChatGPT](https://chatgpt.com)、[Claude](https://claude.ai) 或任何支持的平台
2. 看到聊天框旁边的 **✨** 按钮了吗？
3. 点它 → 浏览 100 条内置提示词 → 选一个 → **砰，它已经在聊天框里了**

恭喜，你已经在用 Prompt Ark 了。

## 刚才发生了什么？

- **✨ 按钮** = **提示词选择器** — 你的提示词库，永远只需一次点击
- **⚡ 按钮** = **快捷指令** — 一键润色、翻译、总结……
- 这两个按钮会 **自动出现** 在所有支持的 AI 平台上

## 下一步做什么？

## AI 功能怎么做到零配置的？

Prompt Ark 默认用 **Gemini Web** 作为 AI 后端。只要你的浏览器登录了 [gemini.google.com](https://gemini.google.com)：

| 功能 | 能用吗？ | 你要做什么 |
|---|---|---|
| ✨ 提示词选择器 | ✅ 始终可用 | 直接点 |
| ⚡ 快捷指令 | ✅ 始终可用 | 直接点 |
| 🔮 AI 优化 | ✅ 默认可用 | 登录 Gemini 就行 |
| 🏷️ 自动标题/分类/标签 | ✅ 默认可用 | 保存时不填标题即可 |

> 想用更快的模型或特定 API？看 [模型配置指南](02-model-configuration.md)。
