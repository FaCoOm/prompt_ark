# 同步、数据与设置

## 选哪个同步方式？

| 方式 | 配置难度 | 容量 | 适合谁 |
|---|---|---|---|
| **Chrome 同步** | 零（默认） | ~100KB | 个人用、小型库 |
| **GitHub Gist** | 粘贴一个 Token | 无限 | 重度用户、团队协作、大型库 |
| **WebDAV** | 填服务器地址 + 密码 | 取决于服务器 | 注重隐私、私有部署（坚果云等） |

## Chrome 同步（默认）
装完自动启用。同一 Google 账号的 Chrome 自动漫游你的提示词。

> ⚠️ Chrome Sync 总容量约 ~100KB。如果装不下了，切 Gist 或 WebDAV。

## GitHub Gist 同步
1. **设置 → 同步** → 选 **GitHub Gist**
2. 粘贴 **GitHub Token**（需要 `gist` 权限）
3. Gist ID 留空——首次同步自动创建
4. 点 **立刻同步 Gist**

## WebDAV 同步
1. **设置 → 同步** → 选 **WebDAV**
2. 填 **WebDAV 地址**、**账号**、**应用密码**
3. 点 **立刻同步 WebDAV**

> 主流 WebDAV 服务：坚果云、Nextcloud、群晖

## JSON 导出 / 导入
- **导出**：设置齿轮 → 导出 → 下载 `.json` 文件
- **导入**：粘贴（原始文本/JSON）或 URL（从网址抓取）

## GitHub Token 有什么用？

在 设置 → 同步 中配置的 Token 解锁三件事：

| 能力 | 没有 Token | 有 Token |
|---|---|---|
| Gist 同步 | ❌ | ✅ |
| 提示词分享 | ❌ | ✅ |
| GitHub URL 导入 | 60 次/小时（容易触发限制） | **5000 次/小时** |

## 自动保存
所有设置页（全部 Tab）都是 600ms 防抖自动保存，无需任何「保存」按钮。仅添加/编辑模型供应商时需要点「保存」按钮确认。
