## 🚀 极速上手

### 第一步：安装与固定
1. 下载或克隆本仓库 → 打开 `chrome://extensions/`（或 `edge://extensions/`）→ 开启**开发者模式** → 点击**加载已解压的扩展程序** → 选择项目文件夹
2. 点击工具栏 **🧩 拼图图标** → 找到 Prompt Ark → 点 **📌 固定**

> 💡 也支持 **侧边栏模式** — 点击图标直接停靠到浏览器侧边常驻使用。

### 第二步：零配置，装好即用
插件默认使用 **Gemini Web** 作为 AI 后端。只要你的浏览器登录了 [gemini.google.com](https://gemini.google.com)，所有 AI 功能**立刻可用**：

| 功能 | 状态 | 你需要做什么 |
|---|---|---|
| ✨ 提示词选择器 | ✅ 始终可用 | 直接点击 |
| ⚡ 快捷指令（润色、翻译……） | ✅ 始终可用 | 直接点击 — 用平台自身的 AI |
| 🔮 AI 提示词优化 | ✅ 默认可用 | 登录 Gemini 即可 |
| 🏷️ 自动生成标题/分类/标签 | ✅ 默认可用 | 保存时不填标题即触发 |

> 💡 不需要 API Key。不需要设置向导。装好，开用。
>
> 想用更快的模型或特定 API？进入 **设置 → 模型 → +** 添加供应商（Gemini API / OpenAI 兼容）。详见 [模型配置指南](docs/zh/02-model-configuration.md)。

### 第三步：创建你的第一个提示词
1. 点击工具栏 **Prompt Ark 图标**，打开管理面板
2. 点击蓝色 **+ 新建** 按钮
3. 写提示词内容 — 用 `{{主题}}` 或 `{{语言}}` 创建**动态变量**
4. 点 **保存**。标题留空的话，AI 会自动生成标题、分类和标签！
5. 点 **预览** 检查 Markdown 渲染效果

### 第四步：在 AI 平台里使用
打开 [ChatGPT](https://chatgpt.com)、[Claude](https://claude.ai)、[Gemini](https://gemini.google.com) 或任何支持的平台，你会看到聊天框旁多了按钮：

| 操作方式 | 做什么 |
|---|---|
| **✨ 选择器** | 浏览提示词库 → 选一个 → 自动注入聊天框 |
| **⚡ 快捷指令** | 一键润色 / 总结 / 翻译 / 扩写 / 解释 |
| **`/斜杠`** | 聊天框输入 `/关键词`，原地展开提示词 |
| **`Ctrl+Shift+P`** | 全局快捷键唤出悬浮选择器 |
| **右键菜单** | 选中文字 → "添加到 Prompt Ark" 瞬间入库 |

### 第五步：多设备同步（可选）
进入 **设置 → 同步**，选择同步引擎：

| 方式 | 适合 |
|---|---|
| **Chrome 同步**（默认） | 零配置、自动同步、~100KB 限制 |
| **GitHub Gist** | 无限容量、团队共享 |
| **WebDAV** | 私有部署、注重隐私（如坚果云） |

> 📚 **[完整文档 →](docs/zh/)** 查看所有功能的详细说明。

## ⚡ 进阶玩法

### 一键文字加工
聊天框旁还有一个 **⚡** 按钮。点它可以一键：

**✏️ 润色** · **📋 总结** · **➕ 扩写** · **🌐 翻译** · **💡 解释**

> 不需要 API Key——直接用平台自身的 AI 处理。

### 动态变量
在提示词里写 `{{语言}}` 或 `{{主题}}`，使用时自动弹出填写表单。每次用都能微调，不用手动改。

### 斜杠命令
在任何聊天框输入 `/邮件` → "商务邮件写手"提示词瞬间展开。像输入法短语扩展，但用于 AI。

### AI 提示词优化器
在编辑弹窗点 ✨ 优化 → 生成 3 个专业变体（精简版 / 增强版 / 专业版），附带逐行 Diff 对比。一键采纳。

### 右键即存
看到网页上一段精彩提示词？选中文字 → 右键 → **"添加到 Prompt Ark"**。自动生成标题、分类和标签。

### 页面感知提示词
在提示词中使用 `{{page_title}}`、`{{selected_text}}`、`{{page_url}}`，它们会自动填充当前页面的内容。
