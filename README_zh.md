## 🚀 极速上手

### 第一步：安装与固定
1. 下载或克隆本仓库 → 打开 `chrome://extensions/`（或 `edge://extensions/`）→ 开启**开发者模式** → 点击**加载已解压的扩展程序** → 选择项目文件夹
2. 点击工具栏 **🧩 拼图图标** → 找到 Prompt Ark → 点 **📌 固定**

> 💡 也支持 **侧边栏模式** — 点击图标直接停靠到浏览器侧边常驻使用。

### 第二步：零配置，装好即用
插件默认使用 **Gemini Web** 作为 AI 后端。只要你的浏览器登录了 [gemini.google.com](https://gemini.google.com)，所有 AI 功能**立刻可用**：

| 功能 | 状态 | 你需要做什么 |
|---|---|---|
| ✨ 提示词选择器 | ✅ 始终可用 | 直接点击 |
| ⚡ 快捷指令（润色、翻译……） | ✅ 始终可用 | 直接点击 — 用平台自身的 AI |
| 🔮 AI 提示词优化 | ✅ 默认可用 | 登录 Gemini 即可 |
| 🏷️ 自动生成标题/分类/标签 | ✅ 默认可用 | 保存时不填标题即触发 |

> 💡 不需要 API Key。不需要设置向导。装好，开用。
>
> 想用更快的模型或特定 API？进入 **设置 → 模型 → +** 添加供应商（Gemini API / OpenAI 兼容）。详见 [模型配置指南](docs/zh/02-model-configuration.md)。

### 第三步：创建你的第一个提示词
1. 点击工具栏 **Prompt Ark 图标**，打开管理面板
2. 点击蓝色 **+ 新建** 按钮
3. 写提示词内容 — 用 `{{主题}}` 或 `{{语言}}` 创建**动态变量**
4. 点 **保存**。标题留空的话，AI 会自动生成标题、分类和标签！
5. 点 **预览** 检查 Markdown 渲染效果

### 第四步：在 AI 平台里使用
打开 [ChatGPT](https://chatgpt.com)、[Claude](https://claude.ai)、[Gemini](https://gemini.google.com) 或任何支持的平台，你会看到聊天框旁多了按钮：

| 操作方式 | 做什么 |
|---|---|
| **✨ 选择器** | 浏览提示词库 → 选一个 → 自动注入聊天框 |
| **⚡ 快捷指令** | 一键润色 / 总结 / 翻译 / 扩写 / 解释 |
| **`/斜杠`** | 聊天框输入 `/关键词`，原地展开提示词 |
| **`Ctrl+Shift+P`** | 全局快捷键唤出悬浮选择器 |
| **`Ctrl+Shift+G`** | **抓取上下文** — 快照当前页面（标题、URL、选中文字）供提示词变量使用 |
| **右键菜单** | 选中文字 → "添加到 Prompt Ark" 存档 · "智能转换" 变成可复用提示词 |

### 第五步：多设备同步（可选）
进入 **设置 → 同步**，选择同步引擎：

| 方式 | 适合 |
|---|---|
| **仅本地**（默认） | 零配置、当前设备使用 |
| **WebDAV** | 私有部署、注重隐私（如坚果云） |
| **Obsidian Vault（WebDAV）** | Markdown 文件工作流、Obsidian 用户 |

> 📚 **[完整文档 →](docs/zh/)** 查看所有功能的详细说明。

## ⚡ 进阶玩法

### 一键文字加工
聊天框旁还有一个 **⚡** 按钮。点它可以一键：

**✏️ 润色** · **📋 总结** · **➕ 扩写** · **🌐 翻译** · **💡 解释**

> 不需要 API Key——直接用平台自身的 AI 处理。

### 动态变量 — 三种类型
在提示词中写变量，使用时自动弹出对应的填写控件：

| 写法 | 类型 | 示例效果 |
|---|---|---|
| `{{主题}}` | 自由输入 | 文本框 |
| `{{语言:中文|英文|日文}}` | 枚举下拉 | 三选一 |
| `{{风格:正式}}` | 预填默认 | 可编辑，已预填 |

### 上下文变量（`{{@...}}`）
用 `{{@page_title}}`、`{{@page_url}}`、`{{@selection}}`、`{{@date}}` 自动填充当前标签页的实时内容——不用手动复制粘贴。

> **抓取上下文**（`Ctrl+Shift+G`）预先快照页面，切换标签页后上下文变量依然有效。

### 智能转换
网页上选中任意文字 → 右键 → **"智能转换为提示词"**。AI 自动推断意图，添加角色设定，提取变量，结构化成可复用提示词——无需手动编辑。

### 一键分享到社交平台
 AI 生成内容，一键分享到各平台：
**Twitter** · **Reddit** · **知乎** · **微信** · **小红书**

### 斜杠命令
在任何聊天框输入 `/邮件` → "商务邮件写手"提示词瞬间展开。像输入法短语扩展，但用于 AI。

### AI 提示词优化器
在编辑弹窗点 ✨ 优化 → 生成 3 个专业变体（精简版 / 增强版 / 专业版），附带逐行 Diff 对比。一键采纳。

### 右键即存
看到网页上一段精彩提示词？选中文字 → 右键 → **"添加到 Prompt Ark"**。自动生成标题、分类和标签。

### 页面感知提示词
在提示词中使用 `{{@page_title}}`、`{{@page_url}}`、`{{@selection}}`、`{{@page_text}}`，自动填充当前页面的实时内容。

## 🛠 面向开发者

- **Manifest V3** — `storage`、`contextMenus`、`scripting`、`sidePanel`、`cookies`
- **深层 DOM 穿透** — Shadow DOM、React Fiber 绕过、平台级注入适配
- **零配置 AI** — Gemini Web 会话逆向工程为默认后端，Edge 显式 Cookie 兼容，会话过期自动跳转登录
- **多供应商 AI** — Gemini API · OpenAI 协议 · Gemini Web（免费），统一调度，运行时热切换
- **提示词文件化** — 所有 LLM 提示词存于 `prompts/*.md`，无需改 JS 即可编辑
- **上下文抓取器** — `Ctrl+Shift+G` 快照页面（10 分钟 TTL），跨标签 `{{@变量}}` 解析
- **智能转换** — 元提示词将选中文字逆向工程为结构化、含变量的可复用提示词
- **Runtime 国际化** — 中英文运行时热切换
- **无感保存** — 600ms 防抖全局自动保存
