## 中文 AI 写作痕迹检测

以下规则在 humanize.md 的 8 条基础改写规则之上追加，专用于**中文**文本。两套规则叠加使用，不互相替代。

### 高频 AI 词汇（命中即删或替换）
此外、值得注意的是、不可否认、总体而言、毋庸置疑、在某种意义上、至关重要、深入探讨、不断演变的格局、关键时刻、里程碑、为……奠定基础、标志着、象征着、体现了、未来可期、拭目以待、充满活力的、丰富的（比喻义）、令人叹为观止、坐落于、位于……的中心

### 系动词回避
AI 用复杂结构代替"是/有"：
- "作为……的体现" → "是"
- "充当/扮演/被视为" → "是"
- "拥有（夸张用法）/设有/提供" → "有"

### 伪口语化高频词（命中即删）
拆一拆、拆开来看、掰开、掰开来看、盘一盘、聊一聊、捋一捋、盘点一下、划重点、敲黑板 — 这类"动词+一+动词"是 LLM 模拟亲切感的统计高频输出

### 降维引导语（全文合计 ≤ 1）
说白了、本质上（就是）、归根结底、一句话概括、简单来说、换个角度看、从某种意义上说、不妨这样理解

### 高频分析动词（全文合计 ≤ 1，优先改为 看/说/讲 或省略）
拆解、梳理、剖析、解构、聚焦、洞察、深耕、赋能、助力、践行、驱动、构建、打造

### 伪学术腔（命中即改）
- "场/场域"化合物：舆论场、话语场、权力场、公共场域 → "舆论"、"讨论"
- "叙事"做万能标签：宏大叙事、官方叙事、主流叙事 → "说法"、"口径"
- 管理咨询体：底层逻辑、顶层设计、认知升级、降维打击、赛道、红利、抓手、闭环、存量、增量、博弈（非博弈论语境）、路径依赖（非经济学语境）

### 戏剧化揭露修辞（正文清零）
遮羞布、最后一块遮羞布、面具、画皮、伪装、外衣、幌子、烟幕弹 — 用于"揭露"框架时命中即改。扯下/撕下/揭下/掀开 + 遮蔽物 → 直接给事实和判断

### 极值判断句式（全文合计 ≤ 1）
最残酷的地方在于、最可怕的地方是、最讽刺的地方在于、真正可怕的是、真正残酷的是、讽刺之处在于、更残酷的是 → 删极值框架，直接铺事实

### 关联句式堆叠（全文合计 ≤ 2）
一旦……就、只有……才、无论……都、随着……的发展/深入/推进、正是因为……所以、通过……来 → 改为散装条件词（要是/若/倘/靠/凭）或直接铺事实

### 二分对照壳（正文清零）
不是A而是B、并非A而是B、不在于A而在于B → 改为 "A固然……；但/而……" 或直接下判断

### 步骤承接过渡（命中即删前半句）
把X说完/拆完/看完/理完/弄清/厘清，再/便/就Y → 删掉前半句，直接从新事实或判断起笔。"先看A""接着看B""上面说的是X，下面说Y" → 直接写 B 的内容

### 鸡汤式收尾（命中即改）
未来可期、我们有理由相信、光明前景、这只是开始、激动人心的时代 → 用冷判断、事实或引文收束

### 标点约束
- 全角冒号（：）≤ 2，且仅用于直接引语提示语（"他说：……"）
- 破折号（——）≤ 3
- 禁用表情符号
- 统一中文引号与全角标点，禁止弯引号与半角标点混用
